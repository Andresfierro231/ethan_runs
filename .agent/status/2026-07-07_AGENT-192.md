## Acceptance Criteria

- Candidate windows are tagged as `stationary`, `quasi_stationary`,
  `moving_not_plateaued`, `oscillatory_not_steady`,
  `short_or_early_terminated`, or `transient_model_only`.
- Each output row carries drift, peak-to-peak, autocorrelation/effective sample
  size, block means, oscillation envelope/period where detectable, uncertainty
  components, total temporal uncertainty, fit-use status, and
  `independence_group_id`.
- Coherent oscillations cannot shrink to tiny standard errors by counting every
  solver write as independent.
- Multiple windows from one relaxation path share one independence group.
- Corrected Salt special-gate rows carry `needs_special_gate_scrutiny=True` and
  are not closure-fit admissible without coordinator review.
- Synthetic tests cover stationary correlated noise, monotonic drift,
  persistent oscillation, short termination, pass-through target, special gate,
  and independence grouping.
