## Raw Observations Vs Interpretation

Raw observations:

- The existing deep-dive note already warned that transient samples are not
  independent steady closure data.
- `tools/analyze/assess_time_convergence.py` already computes drift,
  peak-to-peak amplitude, lag-1 autocorrelation, effective sample size, and an
  autocorrelation-corrected standard error.

Interpretation:

- Those metrics are a good baseline but insufficient for journal-grade temporal
  UQ when coherent oscillations exist. A future worker should add block means,
  oscillation envelope/period diagnostics, explicit window states, conservative
  temporal uncertainty, and independence grouping.
