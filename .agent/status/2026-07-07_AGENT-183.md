**Root cause of second crash:** `ModelResult` (solver.py:334) has no `loop_mean_T_K`
attribute. Available temperature fields are: `start_temperature_K`, `end_temperature_K`,
`max_bulk_temperature_K`, and `segment_states: List[SegmentState]` where each
`SegmentState` has `T_avg_K`, `T_in_K`, `T_out_K`, `s_start_m`, `s_end_m`.

The script comment says `CFD_LOOP_T` is "approx mean bulk T from momentum budget".
The physically consistent 1D proxy is a **length-weighted mean of segment T_avg_K**:
```
loop_mean_T_K = Σ(seg.T_avg_K × (seg.s_end_m - seg.s_start_m)) / Σ(seg.s_end_m - seg.s_start_m)
```
