## Key Finding

The project has most of the infrastructure needed to run a forward model, but
not enough admitted predictive inputs to claim mdot and sensor-temperature
predictivity yet.

Available now:

- geometry and material-property infrastructure,
- Fluid pressure/temperature solve skeleton,
- heater/cooler setup metadata,
- patch-level external `h/Ta/Tsur/emissivity/layer` metadata,
- sensor prediction infrastructure,
- fixed-mdot and parity evidence.

Major blockers:

- cooler/HX predictive duty,
- heater transfer efficiency,
- test-section source-plus-loss model,
- wall-layer external driving temperature,
- radiation inseparability,
- thermal mesh/time-window gates,
- hydraulic fit-safe gates,
- provisional sensor coordinates,
- train/validation split and parameter identifiability.
