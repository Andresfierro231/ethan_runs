# Endpoint Harvest Methods Note

Retained coarse paths were repaired to the June 30 OF13 reconstructions used by the prior section-pressure evidence. Existing raw `.xy` station planes were masked to a 0.04 m radius around each mesh station and evaluated using the expected branch-flow normal.

Because the `.xy` format omits polygon face area, `RAF` is reported as a reverse-count fraction and `RMF` as an equal-face mass-flux proxy. These quantities are useful diagnostic screening evidence but are not ordinary-flow admission evidence.

No scheduler launch, native CFD mutation, registry mutation, F6 fit, or component-K admission was performed.
