# Transient And ROM Parking Lot

Decision: `mf05_mf06_parked_future_only_no_implementation`.

MF-05 and MF-06 remain useful literature-derived future lanes, but neither is a
near-term thesis unlock for the current steady 1D predictive path.

MF-05 should only be reopened if the data show a repeatable positive-mean
periodic fitting response with unchanged topology. It is not appropriate for
startup transients, zero-crossing flow, chaotic oscillation, persistent exchange
cells, or recirculation topology changes.

MF-06 should only be reopened after a verified full-order CFD/FOM snapshot
ensemble exists with boundary metadata, retained-mode or energy criteria,
conservation checks, extrapolation detection, and held-out validation. Existing
ROM work remains useful context, not an admitted thesis model form.

The active thesis path stays steady-state and source-bounded:
setup-only runtime contract, train-only solve, attribution, freeze, validation,
holdout, external-test.
