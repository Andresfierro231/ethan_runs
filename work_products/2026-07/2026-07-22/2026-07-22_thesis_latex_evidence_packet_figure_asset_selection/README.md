# Figure Asset Selection Evidence Packet

This packet lists thesis-ready or near-ready figure assets for external writers
without copying binary figures into the thesis repo.

Files:

- `figure_candidate_ledger.csv`: exact source paths, sizes, target sections,
  and claim boundaries.
- `copy_decision_table.csv`: which assets are good copy candidates later and
  which should remain path-only.
- `external_writer_brief.md`: short instructions for using the figures.

No binary assets are imported in this task.
