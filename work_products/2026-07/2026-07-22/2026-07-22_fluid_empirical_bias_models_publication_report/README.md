# Empirical Bias Models Publication Report

This package writes a numerical-scientific report for the empirical temperature
bias/correction models. It uses only completed diagnostic artifacts and does
not run new solves, fit new coefficients, score new protected rows, or admit a
predictive model.

Open `report.md` first.

## Files

- `report.md`: publication-facing narrative with equations, results,
  interpretation, assumptions, caveats, and next studies.
- `model_family_publication_table.csv`: concise model-family table.
- `claim_boundary_table.csv`: what can and cannot be claimed.
- `follow_on_studies.csv`: discrepancy-identification studies needed before
  admission.
- `source_manifest.csv`: source paths and read-only use.
