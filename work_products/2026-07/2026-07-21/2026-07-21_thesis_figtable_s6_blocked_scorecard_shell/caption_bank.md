# Caption Bank

## Blocked Scorecard Shell

The final scorecard is intentionally shown as a blocked shell. No runtime-legal final candidate has been frozen, no row is released for fitting or model selection, and protected holdout/external/future rows remain score-only after a future freeze.

## S0-S11 Gate Flow

The gate flow separates completed diagnostic/release infrastructure from trigger-gated figure packages. A zero-score table is the current rigorous result because publishing numerical accuracy before freeze would violate the split and source/property gates.

## Split-Role Table

Rows list partition role, source/property status, and release state. Empty score cells are data discipline, not missing formatting.
