# CSEM Current Thesis Draft Packet

## Purpose

This package is the handoff for reading or importing the current CSEM thesis
material. It does not edit the external dissertation repository. It collects
the ready `ethan_runs` thesis-dossier sections and makes the remaining
trigger-gated refreshes explicit.

## Packet Files

- `ordered_reading_manifest.csv`: recommended section order and import target.
- `reviewer_checklist.md`: checks for overclaiming, runtime leakage, source
  paths, and blocked science.
- `trigger_gated_refresh_ledger.csv`: deferred thesis refreshes and required
  evidence before they may be written.

## Current Draft Spine

Read or import the sections in this order:

1. Ch. 1 motivation and contribution.
2. Ch. 3 CFD evidence/database.
3. Ch. 5 steady `fluid+walls` model form.
4. Ch. 6 closure admission and uncertainty.
5. Ch. 7 pressure, thermal, and predictive-path results.
6. Ch. 8 limitations, SAM relevance, and conclusions.
7. LitRev incorporation bridge for chapter-level source-envelope and
   model-form insertions.
8. Figure/table incorporation ledger.

## Current Use Boundary

This packet is ready for internal thesis review and external-manuscript import
planning. It is not a final CSEM dissertation edit and does not change closure
admission state.

Do not claim final predictive scores, admitted passive wall/test-section
closure, ordinary lower-right corner component `K`, F6 recorrection, ordinary
upcomer single-stream coefficients, SAM validation, or Salt-versus-Water
synthesis from this packet.
